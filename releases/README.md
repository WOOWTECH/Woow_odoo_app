# Releases

Download the latest APK from GitHub Actions:

## Latest Build

Go to [Actions](https://github.com/WOOWTECH/Woow_odoo_app/actions) → Select latest successful build → Download `woowtech-odoo-debug` artifact.

## Direct Download

After each successful build, the APK is available as a GitHub artifact for 30 days.

## Installation

1. Download `woowtech-odoo-debug.zip`
2. Extract `app-debug.apk`
3. Enable "Install from unknown sources" on your Android device
4. Install the APK
