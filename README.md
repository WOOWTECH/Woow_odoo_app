# WoowTech Odoo

Android mobile application for accessing Odoo ERP system.

## Features

- **Multi-account Management** - Store and switch between multiple Odoo accounts
- **WebView Integration** - Display Odoo mobile web interface
- **Dynamic Theming** - Customizable app color via color picker
- **Security Options** - Optional biometric and PIN authentication
- **Bilingual Support** - Traditional Chinese and English

## Requirements

- Android 10+ (API 29+)
- Odoo 18 Community server
- HTTPS connection required
